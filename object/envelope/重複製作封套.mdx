title: "重複製作封套"
---

<code>物件</code> <code>封套</code> <code>批次處理</code>

📌 <code>選單路徑：物件 \> 批次編輯 \> 重複製作封套</code>

### 1. 功能

將同一物件分別套用至多個指定的封套形狀中。

### 2. 使用時機

* 同一圖形、插畫或文字需呈現在不同造型中。  
* 製作系列造型或多版本視覺時。  
* 嘗試不同封套形狀進行視覺比較時。

### 3. 基本操作方式

將被套用的物件移置最下方，全選所有物件，在【物件 > 批次編輯】中選取「重複製作封套」。

### 【以最上層物件製做封套】與【重複製作封套】之差異

| 項目 | 以最上層物件製作封套 | 重複製作封套 |
| :---- | :---- | :---- |
| **封套來源** | 最上層物件作為封套形狀 | 多個指定形狀作為封套 |
| **被套用的內容** | 其他選取的物件 | 同一個物件 |
| **內容所在層級** | 依原物件順序 | 最下層 |
| **主要用途** | 用一個形狀，統一變形多個物件 | 用多個形狀，套用同一個內容 |
| **操作概念** | 一對多 ( 1 個形狀 → 多個物件 ) | 多對一 ( 多個形狀 → 1 個內容 ) |

📍 快速選擇指引
<Note>
我有一個形狀，想要套用到很多物件上 ➡ 以最上層物件製作封套  
</Note>

<Note>
我有一個物件，要放進很多不同形狀裡 ➡ 重複製作封套
</Note>
